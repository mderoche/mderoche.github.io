Validate Something

Validicious is a data validator; give it input—strings, integers, objects, functions, your diploma—and it'll tell you if it's valid according to some rules you define. Use pre-built rules like min-length, has-alpha, and credit-card, or write your own. After all rules have been executed against your input, Validicious will tell you which rules failed, and which rules passed.

Asynchronously Scrumptious

Rules can be synchronous or asynchronous; Validicious will take care of the broken promises and callbacks that nobody ever wants to deal with in asynchronous development.

Quick Glances

Asynchronous

Vds.async([ { context: $('#password'), alias: 'password', rules: [{ rule: 'min-length', args: { len: 4 } }, { rule: 'has-alpha' }] } ]).then(function (summary) { console.log(summary.result); });

Synchronous

var validate = Vds.sync([ { context: $('#password'), alias: 'password', rules: [{ rule: 'min-length', args: { len: 4 } }, { rule: 'has-alpha' }] } ]); console.log(validate);

Custom Rule

Vds.Rules.add('username-available', function (rule, context, options) { $.get('/rest/usernames/' + context, function (data) { rule.finish((data.username !== ''), data); }); });

Quick Validation

Vds.sync($('#cost').val(), 'currency', { allowDecimal: true });
