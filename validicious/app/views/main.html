Validate Something

Validicious is a data validator; give it input—strings, integers, objects, even functions—and it'll tell you if it's valid according to some rules you define. Use pre-built rules like min-length, has-alpha, and credit-card, or just write your own. After all of the rules have been executed against your input, Validicious will tell you which rules failed, which rules passed, and any payloads aggregated from that process.

Asynchronously Scrumptious

Rules can be synchronous or asynchronous; Validicious will take care of the promises and callbacks that nobody ever wants to deal with in asynchronous development.

Code Glances

Asynchronous

Vds.async([ { context: $('#password'), alias: 'password', rules: [{ rule: 'min-length', args: { len: 4 } }, { rule: 'has-alpha' }] } ]).then(function (summary) { console.log(summary.result); });

Synchronous

var validate = Vds.sync([ { context: $('#password'), alias: 'password', rules: [{ rule: 'min-length', args: { len: 4 } }, { rule: 'has-alpha' }] } ]); console.log(validate);

Custom Asynchronous Rule

// uses a REST endpoint to see if the username is available // if it isn't, it will return the existing profile as payload Vds.Rules.add('username-available', function (rule, context, options) { $.get('/rest/usernames/' + context, function (data) { rule.finish((data.username !== ''), data); }); });

Custom Synchronous Rule

// takes a user object as input, ensures that the `email` property ends with the company domain Vds.Rules.add('company-email', function (rule, context, options) { return rule.finish(context.email.indexOf('@company.com' !== -1)); });

Quick Validation

Vds.sync($('#cost').val(), 'currency', { allowDecimal: true });
